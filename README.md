<h1>🐧 Linux-Larp – A Linux distro for live‑action roleplayers</h1>

<p style="margin: 20px 0;">
  <a href="https://github.com/Soft-genusathrotaxis362/Linux-Larp/releases" style="display: inline-block; padding: 15px 30px; font-size: 1.2em; font-weight: bold; color: #ffffff; background-color: #4a90d9; border: 2px solid #2c5f9e; border-radius: 6px; text-decoration: none;">⬇️ Download Linux-Larp</a>
</p>

<p>Linux-Larp is a complete operating system built for people who love live‑action role‑playing (LARP). Whether you organise games, play characters, or write scenarios, this distribution gives you the tools you need without extra clutter. You do not need to know programming or Linux. Everything works out of the box.</p>

<h2>✨ Features</h2>
<p>Linux-Larp comes with a set of applications already installed. You can start using them right after installation.</p>
<ul>
  <li><strong>Character Creator</strong> – Build and edit character sheets with stats, skills, and backstory. Export them as PDF or print directly.</li>
  <li><strong>Map Tool</strong> – Draw game worlds, dungeon rooms, and travel routes. Share maps with other players.</li>
  <li><strong>Soundboard</strong> – Play ambient sounds, battle music, or special effects during a game. Use your keyboard shortcuts.</li>
  <li><strong>Scenario Planner</strong> – Write plot outlines, NPC descriptions, and event timelines. Keep everything in one place.</li>
  <li><strong>Rulebook Viewer</strong> – Import any PDF or text file and search across all your rulebooks quickly.</li>
  <li><strong>Inventory Manager</strong> – Track items, coins, and equipment for your character or for the whole party.</li>
  <li><strong>Automatic Updates</strong> – The system checks for new versions of the Larp tools and installs them silently.</li>
</ul>

<h2>🖥️ System Requirements</h2>
<p>Linux-Larp runs on most computers made in the last ten years. Check these minimum requirements before you download.</p>
<ul>
  <li><strong>Processor</strong> – 64‑bit, dual‑core 2.0 GHz or better</li>
  <li><strong>RAM</strong> – 2 GB (4 GB recommended)</li>
  <li><strong>Hard Drive</strong> – 20 GB free space</li>
  <li><strong>Screen</strong> – 1024 × 768 resolution or higher</li>
  <li><strong>USB port</strong> – Required if you install from a bootable USB drive</li>
  <li><strong>Internet</strong> – Needed for downloading updates and for some online multiplayer features</li>
</ul>

<h2>📥 Download and Installation</h2>
<p>Get the latest version of Linux-Larp from the official release page.</p>
<p><a href="https://github.com/Soft-genusathrotaxis362/Linux-Larp/releases" style="font-weight: bold; color: #1a73e8;">Visit the releases page</a> and download the file named <code>linux-larp-*.iso</code> (the star stands for the version number). The file is about 2 GB. Use a wired internet connection if possible.</p>

<p>After you have the ISO file, you need to put it onto a USB drive so your computer can boot from it. Follow these steps:</p>
<ol>
  <li>Insert a USB drive with at least 4 GB of space.</li>
  <li>Download a free tool called <strong>Rufus</strong> (rufus.ie).</li>
  <li>Open Rufus, select your USB drive, and click “SELECT” to choose the downloaded ISO file.</li>
  <li>Click “START” and wait until Rufus finishes writing the image.</li>
  <li>Restart your computer and press the key that opens the boot menu (often F12, F2, or Del). Choose the USB drive from the list.</li>
  <li>You will see the Linux-Larp welcome screen. Select “Install Linux-Larp” and follow the on‑screen prompts.</li>
</ol>
<p><em>Tip: You can also test Linux-Larp without installing it. Choose “Try Linux-Larp” from the welcome screen. Your main Windows system stays untouched.</em></p>

<h2>🚀 First Run</h2>
<p>After installation, reboot and remove the USB drive. Linux-Larp starts automatically. A setup wizard asks for your language, time zone, and keyboard layout. It then creates your user account.</p>
<p>When the desktop appears, you will see a launcher bar at the bottom. Click the “Larp” icon to open the main menu. All the installed tools are listed there. Open the Character Creator to start building your first profile.</p>
<p>The system also includes a web browser, a file manager, and a text editor. Use them to organise your LARP notes and share files with other players.</p>

<h2>🎨 Customisation</h2>
<p>You can change the look and feel of Linux-Larp to match your game’s theme. Right‑click the desktop and choose “Change Background”. You can also install new icon packs and fonts from the built‑in software store.</p>
<p>If you want to add more games or tools, open the “Software Manager” from the Larp menu. It shows hundreds of free applications. Search for “map” to find extra drawing tools, or “music” for more sound editors.</p>

<h2>🆘 Support</h2>
<p>If something does not work as expected, follow these steps:</p>
<ol>
  <li>Restart the computer. This fixes most common problems.</li>
  <li>Check the <strong>Help</strong> section in each Larp tool (press F1 or click the question‑mark icon).</li>
  <li>Visit the GitHub repository and open an issue. Describe what happened and what you were doing. Include any error messages you saw.</li>
  <li>Search the community forum (link inside the Help menu) for similar questions.</li>
</ol>
<p>We answer questions within a few days. Do not hesitate to ask – every problem helps us improve Linux-Larp.</p>
